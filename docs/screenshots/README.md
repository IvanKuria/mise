# Screenshots

Drop the README screenshots here. The README references these filenames:

- `notch-collapsed.png` — the resting/collapsed notch.
- `notch-expanded.png` — the expanded panel showing all three sections.
- `recent-detail.png` — a Recently watched poster tapped open to its TMDB
  synopsis detail view.
- `heatmap.png` — the contribution heatmap with the year switcher.

Suggested capture size: retina (2x) PNGs, cropped to the notch panel with a
little surrounding wallpaper so the translucency reads.

When the in-app TMDB attribution is shown, make sure the **TMDB logo** is
visible alongside the text "This product uses the TMDB API but is not endorsed
or certified by TMDB."
